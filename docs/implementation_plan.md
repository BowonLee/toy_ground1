# 구현 계획서 - 이벤트 홍보 랜딩 페이지

## 목표
영화 시사회나 게임 런칭 페이지와 같은 **고품질 비주얼 중심의 홍보용 웹사이트** 제작.
사용자 참여(RSVP)보다는 **정보 전달과 기대감 조성**이 주 목적.

## 기술 스택 (확정)
**React + Tailwind CSS**
- **선정 이유**: 화려한 비주얼과 애니메이션 구현에 적합.

## 프로젝트 구조
- `index.html`: 진입점.
- `src/`
    - `components/`: Hero, About, Details, Footer.
    - `App.jsx`: 전체 레이아웃.
    - `index.css`: Tailwind 및 커스텀 애니메이션.
- `docs/`: 기획 문서.

## 구현 상세 (비주얼 중심)
1.  **디자인 컨셉**:
    -   **Immersive (몰입감)**: 큰 배경 이미지/비디오, 어두운 테마(Dark Mode) 권장.
    -   **Typography**: 타이틀에 강렬한 폰트 사용.
    -   **Motion**: 스크롤에 따른 페이드인, 패럴랙스 효과.

2.  **주요 섹션**:
    -   `Hero`: 화면을 가득 채우는 비주얼, 강렬한 타이틀 로고.
    -   `About/Story`: 모임/이벤트의 배경 설명 (영화 시놉시스처럼).
    -   `Details`: 일시, 장소 정보를 스타일리시하게 표현.
    -   `Footer`: 저작권 및 간단한 링크. (참여 버튼 없음)

## 2. 페이지 구조 및 섹션

### 2.1 전체 레이아웃 (Single Page Application)

```
┌─────────────────────────────────────┐
│  Hero Section (Above the fold)      │
│  - 헤드라인 + 서브헤드라인           │
│  - 이미지 슬라이드 (좌우 스크롤)│
│  - 예고편 동영상             │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│  About/Story                  │
│  - 영화 케치프레이즈, 영화 소개                  │
│  - 평론가 리뷰                  │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│  Details Section                   │
│  - 일시, 장소 정보                │
│  - 준비사항            │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│  Footer                 │
│  - 저작권 정보             │
│  - 간단한 링크         │
└─────────────────────────────────────┘
```


## 검증 계획
- 다양한 디바이스에서의 비주얼 깨짐 확인.
- 이미지 로딩 속도 및 성능 최적화.
