# Frankenstein (2025) Event Page 계획서

## 📌 문서 정보
- **작성일**: 2025-11-23
- **목적**: 영화 프랑켄슈타인 (2025) 시청 모임 홍보 및 정보 전달
- **기반 문서**: implementation_plan.md, content_plan.md

---

## 1. 목표 및 목적

### 1.1 주요 목표
- **기대감 조성**: 고품질 비주얼과 몰입감 있는 디자인으로 영화에 대한 기대감 증폭
- **정보 전달**: 시청 모임의 일시, 장소, 준비물 등 핵심 정보 명확히 전달
- **분위기 연출**: 고딕 SF 스릴러의 독특한 분위기를 웹사이트 경험으로 확장

### 1.2 타겟 오디언스
1. **영화 팬**: 프랑켄슈타인 원작 및 SF/스릴러 장르 팬
2. **이벤트 참여자**: 트레바리 독서 모임 및 영화 시청 모임 참여자
3. **비주얼 탐색자**: 화려한 웹 인터랙션과 디자인을 즐기는 사용자

## 2. 페이지 구조 및 섹션

### 2.1 전체 레이아웃 (Single Page Application)

```
┌─────────────────────────────────────┐
│  Hero Section (Above the fold)      │
│  - 메인 카피 + 서브 카피             │
│  - 배경 슬라이드/비디오              │
│  - 예고편 보기 버튼/링크             │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│  Story Section (About)              │
│  - 시놉시스 및 스토리텔링            │
│  - 분위기 있는 배경/타이포그래피      │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│  Event Details Section              │
│  - 일시, 장소, 준비물                │
│  - 지도 또는 약도 (선택)             │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│  Footer                             │
│  - Copyright                        │
│  - 관련 링크                        │
└─────────────────────────────────────┘
```

### 2.2 각 섹션 상세 계획

#### Hero Section
**목적**: 압도적인 첫인상과 영화의 분위기 전달

**콘텐츠**:
- **메인 카피**: "신이 되려 한 자 괴물이 될지니"
- **서브 카피**: "Frankenstein (2025)"
- **비주얼**: 어둡고 웅장한 실험실 또는 설산 배경 (이미지 3장 슬라이드)
- **미디어**: 예고편 YouTube 링크 연동

**디자인 요구사항**:
- 화면을 가득 채우는 풀스크린 배경
- 강렬하고 거친 질감의 타이포그래피
- 스크롤 시 페이드 아웃 효과

---

#### Story Section (About)
**목적**: 영화의 세계관과 감정선 전달 (시놉시스)

**콘텐츠**:
- **스토리텔링**:
  "신이 되려 했던 인간, 그리고 인간이 되고 싶었던 괴물.
  2025년, 고전의 완벽한 재해석으로 돌아온 <프랑켄슈타인>.
  과학의 오만함이 빚어낸 비극과 그 속에 숨겨진 슬픈 영혼의 이야기를 함께 나눕니다."
- **키워드**: #Gothic #SciFi #Thriller #MovieNight

**디자인**:
- 텍스트 중심의 레이아웃이지만 배경에 은은한 모션 효과 (안개, 입자 등)
- 가독성을 해치지 않는 선에서 고딕 스타일 폰트 활용

---

#### Event Details Section
**목적**: 참여를 위한 실질적인 정보 제공

**콘텐츠**:
- **일시**: 미정 (TBD)
- **장소**: 트레바리 강남 아지트, 시청각실
- **준비물**: 넉넉한 시간, 열린 마음

**디자인**:
- 정보의 명확한 전달을 위한 깔끔한 레이아웃
- 아이콘 활용 (달력, 위치 핀, 시계 등)
- 섹션 구분선 또는 배경색 변화로 주목도 향상

---

#### Footer
**목적**: 저작권 명시 및 마무리

**콘텐츠**:
- **Copyright**: © Shall we act? - since 2023, From Concidence to connection

**디자인**:
- 미니멀하고 차분한 디자인
- 하단에 고정되거나 콘텐츠 끝에 위치

## 3. 디자인 컨셉

### 3.1 컬러 팔레트
- **Primary**: Deep Black & Dark Grey (어둠, 공포)
  - 예: `#0A0A0A`, `#1F1F1F`
- **Secondary**: Blood Red or Electric Blue (생명, 실험, 경고)
  - 예: `#8B0000` (핏빛), `#00FFFF` (전기 스파크)
- **Text**: Off-White (가독성 및 차가운 느낌)
  - 예: `#E0E0E0`, `#F5F5F5`

### 3.2 타이포그래피
- **헤드라인**: 임팩트 있는 세리프 또는 거친 산세리프 (예: Cinzel, Roboto Slab)
- **본문**: 가독성 좋은 산세리프 (예: Pretendard, Roboto)

### 3.3 이미지/비주얼
- **무드**: 고딕, 느와르, SF, 미스터리
- **소스**: 영화 포스터, 스틸컷, 컨셉 아트

### 3.4 애니메이션
- **스크롤**: Parallax (패럴랙스), Fade-in
- **전환**: 부드러운 이미지 슬라이드
- **효과**: 글리치(Glitch) 효과 또는 안개 효과 (선택 사항)
